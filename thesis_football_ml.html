Abstract
The incorporation of machine learning algorithms in various domains has revolutionized prediction systems, achieving unmatched accuracy. By employing feature engineering and exploratory data analysis, a tailored set of features was developed to reveal the crucial factors that influence football match results. The study examines the last 23 seasons of the English Premier League, analyzing and combining actual match data with bookmakers' and betting data. Six models were executed with Grid Search and Randomized Search across their hyperparameter spaces. Logistic Regression topped all models with 96% accuracy — surpassing prior state-of-the-art results achieved using far narrower datasets. The work introduces novel features through feature engineering — streak and form metrics — which proved to have crucial impact on a team's performance as predictors.

03
Hyperparameter optimization
How can hyperparameters be optimized across different strategies? Grid Search vs Randomized Search across 6 models — evaluating which combination of model architecture and tuning strategy produces the highest predictive accuracy.

ML models — technical overview
Logistic Regression
A classification model (not regression) that applies a sigmoid function to a linear combination of features: σ(z) = 1/(1+e⁻ᶻ). Despite its simplicity, it achieved the highest accuracy in this study — a testament to the quality of the engineered features and the linear separability of the processed data.
Support Vector Machine
Constructs an optimal hyperplane that maximizes the margin between classes. Effective in high-dimensional spaces. Grid Search outperformed Randomized Search for SVM, achieving 90% accuracy — the second-highest individual model result.
Random Forest
Ensemble of decision trees trained on bootstrapped samples, combining predictions through majority vote. Despite its popularity in sports prediction literature, it underperformed here (77%) — suggesting that the EPL data's structure favors linear over tree-based decomposition.
AdaBoost
Adaptive Boosting — iteratively trains weak classifiers, assigning higher weight to misclassified samples. Achieved 82% accuracy. Randomized Search underperformed Grid Search for this model, suggesting its hyperparameter space is more structured.
XGBoost
Gradient Boosted Trees with regularization. Highly competitive in structured data tasks. Achieved 84% with Randomized Search. Its performance gap with Logistic Regression confirms that the feature engineering pipeline reduced the complexity advantage of tree-based methods.
Ensemble Voting + SMOTE
Combined predictions of multiple models through hard voting. Applied SMOTE (Synthetic Minority Oversampling) to address class imbalance. The SMOTE variant achieved 91% — the second-best overall — suggesting class balance is a significant factor in this dataset.
// 07
Hyperparameter tuning — Grid vs Randomized
Grid Search
Exhaustively evaluates every combination within a specified parameter grid. Computationally expensive but guaranteed to find the best combination within the defined space. Better results for SVM, Random Forest, and AdaBoost in this study. Optimal when the parameter space is small and well-understood.
Randomized Search
Samples random combinations from parameter distributions. More efficient and covers larger spaces. Better results for Logistic Regression and XGBoost. The key insight: when the optimal hyperparameters aren't predictable from domain knowledge, random sampling often finds superior configurations more efficiently than exhaustive search.

Future research directions
Direction 01
Psychological factors
Incorporate sentiment analysis and player psychology indicators into predictive models. Real-time tracking data and social media signals to capture team morale and public sentiment ahead of matches.
Direction 02
Multi-league expansion
Expand the dataset to cover top European leagues and international tournaments. The current model's limitation in generalizing across leagues is explicitly acknowledged — a cross-league model remains an open problem.
Direction 03
Player-level granularity
Incorporate individual player performance metrics, fitness tracking, and injury probability estimates. Current models operate at team level — player-level features could significantly improve accuracy for matches where lineup changes are decisive.
Direction 04
Managerial intelligence
Integrate managerial records, tactical preferences, and in-game decision-making patterns. A manager change mid-season is known to significantly affect team performance — this signal is currently absent from the model.
Direction 05
Real-time integration
Build a live prediction system that updates predictions in-game using real-time event data. The current model predicts pre-match — real-time integration could produce a dynamic probability model that adjusts as the match unfolds.
Direction 06
Advanced architectures
Explore deep learning approaches — LSTMs for sequential match-form modeling, Transformers for player-interaction graphs. The 96% accuracy baseline set here provides a strong benchmark for these explorations.

Conclusion
A 96% accuracy that reshapes the field.
Previous research in football match prediction achieved a ceiling of approximately 85% accuracy using Random Forest on limited seasonal data. This thesis pushed beyond that boundary by combining three key innovations: a much larger dataset (23 EPL seasons), the integration of bookmakers' odds data alongside traditional match statistics, and carefully engineered features — particularly streak and form metrics — that capture the temporal momentum of teams across a season.

The finding that Logistic Regression outperformed far more complex models is not a failure of complexity — it is a demonstration of what proper feature engineering can accomplish. When the input data is well-structured and the features are meaningful, a linear model's interpretability becomes an advantage rather than a limitation.

The study also demonstrates the real cost of narrow datasets. Prior work based on two seasons or cup events cannot generalize conclusions about EPL football dynamics. Twenty-three seasons of data reveals patterns that are simply invisible in shorter windows.
